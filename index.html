HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head is where all meta-data is stored

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content
  • ". element {" describes what is being changed by the css code
  • Link allows the html code to relate to a file of css code
  • Head is where meta-data is shared and where links to css styles will be linked

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it

Javascript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Array's can store multiple values in a single reference
  • Comments are snippets of text that can be added along with code. The browser ignores text marked as comments. Comments must be placed after two slashes
  • Operators are mathematical symbols that produce a result based on two values
  • Conditionals are code structures used to test if an expression returns as true or not
  • Functions are a way of packaging functionality that you wish to reuse. It's possible to define a body of code as a function that executes when you call the function name in your "code" Good alternative to repeatedly writing the same code
